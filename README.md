# 签签到 — 每日签到考勤系统

一个面向公司日常考勤的团队签到 Web 应用，支持团队管理、每日签到/签退和考勤记录查看。

## 功能

- 🔐 **用户系统** — 注册/登录，支持管理员和员工两种角色
- 👥 **团队管理** — 创建团队、邀请码加入、成员列表
- ✅ **每日签到** — 一键签到/签退，自动判断迟到/早退
- 📊 **考勤记录** — 个人考勤日历、团队全员考勤查看
- 📈 **数据统计** — 出勤天数、正常/迟到/早退统计

## 技术栈

| 层 | 技术 |
|---|---|
| 前端 | HTML + CSS + JavaScript（SPA） |
| 后端 | Python FastAPI |
| 数据库 | SQLite |
| 认证 | JWT + bcrypt |

## 快速开始

### 1. 安装后端依赖

```bash
cd backend
pip install -r requirements.txt
```

### 2. 启动后端

```bash
python main.py
```

后端默认运行在 `http://localhost:8000`

### 3. 打开前端

直接用浏览器打开 `frontend/index.html`，或使用任意静态文件服务器。

## 使用说明

1. **注册账号** — 选择「管理员」或「员工」角色
2. **创建团队**（管理员）— 输入团队名称，获得邀请码
3. **加入团队**（员工）— 输入管理员分享的邀请码
4. **每日签到** — 9:00 前签到为正常，之后为迟到
5. **每日签退** — 18:00 前签退为早退
6. **查看考勤** — 切换月份查看历史记录
